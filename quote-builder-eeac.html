Quote Prompt Generator

Build a structured prompt in seconds. Use form fields or quick-paste boxes — whichever's faster. Paste the result into your PMTP Quote Builder Claude project.

This tool generates a prompt. That prompt goes into a Claude Project you create once — Claude already knows the PMTP brand, design standards, and form setup every time you start a new quote.

The workflow every time: Fill this form → Generate prompt → Copy it → Open your PMTP Quote Builder project in Claude → Start a NEW chat → Paste → Claude builds the full HTML quote. One chat = one client quote. That way every quote is easy to find later.

One-time setup — create your Claude Project

  1. Go to claude.ai and sign in with your paid account (Pro, Team, or Max).
  2. Click Projects in the left sidebar.
  3. Click + New Project — name it exactly: PMTP Quote Builder
  4. Inside the project, click the pencil icon or Project instructions at the top.
  5. Copy the full project instructions below using the Copy button, paste them into that field, and click Save.
  6. Done. Every quote from here forward: open this project → new chat → paste your prompt.

You only do steps 1–6 once. After that, just open the project and start a new chat for each client.

Save the four shortcuts agents use most

        
In your AI extension sidebar, look for a shortcuts or saved prompts setting. Save each of these so you can call them quickly with /hotel, /tour, /flight, /transfer:


        

          
/hotel-extract

          
Read this hotel webpage and extract the following, formatted as plain text I can paste into a quote-building tool. Do NOT include any prices or rates from this page.

Output exactly this structure:

Hotel name: [name]
Location: [neighborhood, city]
Star rating / category: [if shown]
Room categories: [list each room category name only — no prices]
What's included: [breakfast, Wi-Fi, parking, etc.]
Highlights: [3-5 short bullets — pool, spa, restaurants, view, walking distance to X]
Address: [street address]
Website: [the URL of this page]

Keep it tight. Skip marketing fluff. Skip prices entirely.

        


        

          
/tour-extract

          
Read this tour operator page and extract the following, formatted as plain text I can paste into a quote-building tool. Do NOT include any prices or rates from this page.

Output exactly this structure:

Operator: [Trafalgar / Globus / etc.]
Tour name: [full tour name]
Duration: [e.g., 14 days / 13 nights]
Itinerary highlights: [list day-by-day or city-by-city — keep it concise]
What's included: [meals, transportation, accommodations level, guides, included experiences]
What's not included: [optional excursions, tips, flights, etc., if mentioned]
Pricing tiers: [list tier names only — Standard / Premium / Deluxe — no prices]
Source URL: [the URL of this page]

Skip prices entirely. Keep the itinerary section to one line per day or city.

        


        

          
/flight-extract

          
Read this flight search result or booking page and extract every flight option shown. Do NOT include any prices. For EACH distinct option, output one block using this exact structure:

---
Option [N]:
Airline(s): [carrier name(s) — list all if codeshare or connecting]
Number of stops: [Nonstop / 1 stop / 2 stops]

OUTBOUND:
  Segment 1: [Flight number] · [Departure city & airport code] [Departure time] → [Arrival city & airport code] [Arrival time] ([flight duration])
  Layover: [X hr Y min] at [City, Airport Code]   ← include only if there is a stop
  Segment 2: [Flight number] · [Departure city & airport code] [Departure time] → [Arrival city & airport code] [Arrival time] ([flight duration])
  Total outbound travel time: [X hr Y min]
  Outbound date: [Day, Month Date, Year]

RETURN:
  Segment 1: [Flight number] · [Departure city & airport code] [Departure time] → [Arrival city & airport code] [Arrival time] ([flight duration])
  Layover: [X hr Y min] at [City, Airport Code]   ← include only if there is a stop
  Segment 2: [Flight number] · [Departure city & airport code] [Departure time] → [Arrival city & airport code] [Arrival time] ([flight duration])
  Total return travel time: [X hr Y min]
  Return date: [Day, Month Date, Year]

Cabin class: [Economy / Premium Economy / Business / First]
Fare type / tier: [e.g., Basic Economy, Main Cabin, Flexible, Saver, Plus]
Baggage:
  Carry-on: [Included / Not included / 1 personal item only]
  First checked bag: [Free / $X per bag]
  Additional bags: [if shown]
Changes: [Allowed with fee / Free changes / Non-refundable / Varies by fare]
Mileage eligible: [Yes / No / if shown]
Notes: [Anything else worth knowing — overnight flight, arrives next day, departs from alternate airport, etc.]
---

List each available option as a separate block. If nonstop, omit the Layover and Segment 2 lines.
Skip all prices. Do not summarize — give every segment detail shown on the page.

        


        

          
/transfer-extract

          
Read this transfer / private driver / shuttle page and extract the following, formatted as plain text I can paste into a quote-building tool. Do NOT include prices.

Output exactly this structure:

Service: [private transfer / shared shuttle / private driver / etc.]
From → To: [pickup → dropoff locations]
Vehicle type: [if specified]
Capacity: [number of passengers]
Duration: [estimated travel time]
Inclusions: [meet & greet, child seats, water, etc.]
Provider: [company name]
Source URL: [the URL of this page]

Skip prices entirely.

        


        

          
/photos-extract

          
Find every image on this page that shows the actual property, restaurant, tour, or experience (not logos, icons, banner ads, or stock illustrations). For each image, give me only the direct image URL — one per line, no labels, no descriptions, no commentary.

Skip:
- Site logos and brand marks
- Social media icons
- Banner ads or promotional graphics
- Map screenshots
- User profile photos / avatars
- Any image hotlinked from these blocked sources: trvl-media.com, hotels.com, vrbo.com, muscache.com, encrypted-tbn0.gstatic.com (Google Images cache)

Output format — just the URLs, like:
https://example.com/photo1.jpg
https://example.com/photo2.jpg
https://example.com/photo3.jpg

If you find more than 15 usable images, give me the 15 best ones (variety: exterior, rooms, restaurants, amenities, views). Order them most-to-least useful.
